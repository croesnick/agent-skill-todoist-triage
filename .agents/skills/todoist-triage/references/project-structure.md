## Placement Rules

1. **When in doubt, use Inbox** — better to leave unsorted than misfiled
2. **Family ⭐ gets priority** — P1 items in Family are non-negotiable
3. **SOMEDAY MAYBE is for P4 only** — if it might happen this week, it doesn't belong here
4. **Rituals** = recurring habits, not one-off tasks
5. **TechDay** = exploratory/experimental — if it has a deadline, it probably belongs elsewhere
